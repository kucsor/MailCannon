# **App Name**: MailCannon

## Core Features:

- Email Input: Text field to enter the sender's email address.
- Email List Input: A field where the user can paste or add a list of multiple recipient email addresses.
- Message Editor: A rich text editor to create the email message body.
- Attachment Upload: Functionality to upload a CV/resume file to attach to the email.
- Bulk Email Sending: Processes the email list and sends the email with attachment to each address.
- Drafting Tool: A generative AI tool to help improve the text or generate from scratch job applications.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for professionalism and trust.
- Background color: Light gray (#EEEEEE) for clean, easy readability.
- Accent color: Teal (#009688) for highlighting key actions and buttons.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, neutral appearance.
- Use simple, professional icons for email, attachment, and send actions.
- Clean, well-organized layout with clear sections for each function (email, message, attachments, list).
- Subtle loading animations during the sending process.
